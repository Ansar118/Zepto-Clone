# 🛒 Zepto Clone

A responsive front-end clone of the Zepto grocery delivery website built using **HTML**, **CSS**, and **JavaScript**.

## 🎯 Project Goals
- Recreate the Zepto user interface
- Build reusable and responsive components
- Practice modern HTML, CSS, and JavaScript
- Improve front-end development skills
- Upload daily progress to GitHub

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## ✨ Features
- Responsive Navigation Bar
- Hero Section
- Search Bar
- Product Categories
- Product Cards
- Hover Effects & Animations
- Mobile-Friendly Design
- Interactive UI Components

## 📅 Progress
This repository is updated **daily** with new features, UI enhancements, bug fixes, and code improvements as I continue building the project.
